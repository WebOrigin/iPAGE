1, What is the difference between dynamic website and static website?
Dynamic website is also called database driven website. It requires programming to realize the editable function through a content management system. You
(as administrator of the website), can update information such as text and image through the system. Also, some web programs can only work in the dynamic web
environment, such as online shopping and forum.

2, What functions a static website can have?
In fact, a static website can realize many basic functions, such as displaying text, image, flash and even playing music and video.

3, How do I determine on that whether I need a dynamic website or a static website?
The cost of building a dynamic website is higher than a static website, because we also need to build a database and customise our content management system
“IPAGE CMS” for you to maintain the website. However, it can be very cost-effective in the future if you want to update your website as you can do it yourself.
For example, you may want to change a case study. You can upload the new information by yourself through the “IPAGE CMS” system any time you wish.

So, you need to ask how frequent you may need to change your website’s content. If you need to update it regularly like twice a month, for such operation in long
term, a dynamic website will be better than a static website. Also, if you are planning to have a website which can provide certain interactive functions, such as
discussion forum or online shopping, it means that only dynamic website can do.

4, What do I need to concern regarding to the cost of building a website?
1) How many pages?
2) How do I want to display images? (Flash or static presentation)
3) Do I have particular emphasis on graphic design?
4) Do I need any special web function in the website, such as online voting?
5) Do I need a dynamic website or static website?

5, What do I need to supply to IPAGE New Zealand if I decide to build a website?
You firstly need to provide us a brief which include a basic structure of information flow and design requirement. Copies and images in electronic formats are also required.
